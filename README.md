# 🌐 Student Help Hub

A responsive multi-page website designed to provide students with a simple and organized platform to access useful academic resources and support.

## 📌 Project Overview

Student Help Hub is a multi-page web project created to demonstrate the development of a structured and responsive website.

The website includes multiple pages with consistent navigation, responsive layouts, organized content sections, and a contact form for user interaction.

## ✨ Features

- 🏠 Home page
- 👤 About page
- 📚 Resources and services page
- 📩 Contact page
- 📱 Responsive design for desktop and mobile devices
- 🧭 Navigation between multiple pages
- 🎨 Consistent user interface across all pages
- 📝 Contact form with Java backend integration
- 💾 Contact form data saved by the backend
- 📐 Clean and structured website layout

## 🛠️ Technologies Used

### Frontend

- HTML5
- CSS3
- JavaScript

### Backend

- Java
- Java HTTP Server

## 📄 Website Pages

### Home

Introduces the Student Help Hub and provides an overview of the platform.

### About

Provides information about the purpose and vision of the Student Help Hub.

### Resources

Displays useful resources and services available for students.

### Contact

Allows users to submit their name, email, subject, and message through a contact form.

## ⚙️ Java Backend

The contact form is connected to a Java backend server.

The backend:

- Receives contact form data
- Validates the submitted information
- Handles HTTP requests
- Stores submitted messages in a local file

## 🚀 How to Run the Project

### Frontend

1. Open the project folder in VS Code.
2. Open `index.html`.
3. Run the project using the Live Server extension.

### Backend

1. Open the backend folder in the terminal.
2. Compile the Java server:

```bash
javac StudentHelpHubServer.java
